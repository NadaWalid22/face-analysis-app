# Sample Images

Place front-facing portrait images here for local testing.

The demo loads these in `demo/app.py` as example images.

**Requirements for good results:**
- Full face visible, roughly centered
- Frontal or near-frontal pose (±15° yaw/pitch)
- Good lighting, eyes open
- Minimum 200×200 px resolution

**Privacy note:** Do not commit identifiable photos to this repository.
Use synthetic/anonymized test faces (e.g., generated via StyleGAN2 — [thispersondoesnotexist.com](https://thispersondoesnotexist.com)).
